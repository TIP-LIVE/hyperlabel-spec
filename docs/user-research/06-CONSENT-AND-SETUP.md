## Recording Setup Checklist

### Tools
| Tool | Purpose | Free Tier? |
|------|---------|------------|
| **Zoom** | Video call + built-in recording | Yes (40 min limit on free) |
| **Google Meet** | Alternative if Zoom limited | Yes |
| **Grain** (grain.com) | Auto-transcription + highlights | Free for 20 recordings |
| **Otter.ai** | Transcription alternative | Free 600 min/month |
| **tl;dv** | Alternative with Zoom/Meet integration | Free tier available |

### Before Each Call
- [ ] Test microphone and camera
- [ ] Start Zoom/Meet recording (local or cloud)
- [ ] Start transcription tool (Grain, Otter, or tl;dv)
- [ ] Have interview script open on a second screen or tab
- [ ] Have note-taking template open
- [ ] Confirm participant consent verbally at start of call
- [ ] Share screen ready with TIP concept slides

### After Each Call
- [ ] Save recording with naming convention: `YYYY-MM-DD_Persona_FirstName.mp4`
- [ ] Export transcript
- [ ] Fill in post-interview notes within 2 hours
- [ ] Send gift card within 24 hours
- [ ] Send thank-you email
- [ ] Update tracking spreadsheet

### Calendly Setup (recommended)
1. Create a "TIP Research Interview" event type (45 min)
2. Add buffer time: 15 min before and after
3. Set availability to your preferred interview windows
4. Add the consent form text to the event description
5. Add a pre-event question: "What's your role and company?" (helps with prep)
6. Confirmation email should include: Zoom link, consent note, "feel free to reschedule"

# TIP User Research — Starter Kit

## Quick Start

1. **Read the objectives** → `01-RESEARCH-OBJECTIVES.md` (hypotheses to validate)
2. **Set up Calendly** → see `06-CONSENT-AND-SETUP.md` for checklist
3. **Start sourcing** → use templates in `05-OUTREACH-TEMPLATES.md`
4. **Before each call** → open the right script (`02`, `03`, or `04`) + note template from `07`
5. **After each call** → fill summary, send gift card, update tracker

## Files

| File | What It Is |
|------|-----------|
| `01-RESEARCH-OBJECTIVES.md` | 6 hypotheses, research questions, success metrics |
| `02-INTERVIEW-SCRIPT-CONSIGNEE.md` | 45-60 min script for importers/buyers |
| `03-INTERVIEW-SCRIPT-FORWARDER.md` | 45-60 min script for freight forwarders |
| `04-INTERVIEW-SCRIPT-SHIPPER.md` | 45-60 min script for 3PLs/carriers |
| `05-OUTREACH-TEMPLATES.md` | LinkedIn messages, emails, screener questions, sourcing channels |
| `06-CONSENT-AND-SETUP.md` | Consent form, recording tools, Calendly setup |
| `07-TRACKING-AND-TEMPLATES.md` | Tracker spreadsheet, post-interview summary, weekly debrief |

## Target: 15-20 Interviews

| Persona | Target | Focus |
|---------|--------|-------|
| Consignee (Buyer) | 8-10 | Pain points, pricing, features |
| Forwarder | 4-6 | Workflow, activation, partnership |
| Shipper/Logistics | 2-3 | Industry validation |
